# Comentar lineas

El comentar lineas es codigo que no va aparecer a la hora de ejecutar un programa funciona para documentar el codigo e indicar para que funciona la linea usada.

```python
    # Esta es una linea de codigo
    print("Hola mundo") # Imprime hola mundo

    # Comentarios multilineas
    """
        Se abre y cierra con 3 comillas dobles
    """
```
