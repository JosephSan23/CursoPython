# Tipos de datos

Cadenas de texto

```python
comillasSimples = 'Esto es un texto'
comillasDobles = "Esto tambien"
comillasTriples = '''Esto igual'''
```

Numeros

```python
numeroEntero = 1
numeroDecimal = 3.14
numeroComplejo = 5 + 2j
```

Listas, los elementos pueden cambiar mutables

```python
listaOrdenada = [0, 1, 2, 3 , 4, 5]
```

Tuplas, los elementos son inmutables

```python
tupla = ("a", "b", "c")
```

Diccionarios

```python
diccionario = {
    "nombre": "Juan",
    "edad": 30,
    "ciudad": "Madrid"
}
```

Conjuntos, coleccion desordenada, los elementos son unicos

```python
conjunto = {1, 1, 2, 2, 3} #output: {1,2,3}
```

Booleanos, solo son dos valores

```python
true = True
false = False
```
