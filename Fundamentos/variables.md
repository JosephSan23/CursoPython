# Variables

Son unas cajas donde se guarda un valor para poder utilizarlo, esta caja es un lugar de memoria donde python podra consultar para poder recuperar ese valor.

```python
x = "Esta es una variable"
print(x)
```

Hay que tener cuidado con las variables ya que python permite sobreescribir los valores asignados de las variables.

Case sensitive: Se puede escribir variables tanto como mayusculas y minusculas pero se almacenaran en distintos espacios de memoria, esto significa que no se sobreescribira el valor.

```python
x = 1
print(x)
X = 2
print(X)
```

Formas para crear una variable:

```python
mivariable = "Esta es una variable"
mi_variable = "Esta es una variable con guion bajo"
_mi_variable = "Esta variable comienza con guion bajo, cuando las variables con privadas"
miVariable = "Camel case"
MIVARIABLE = "Todo mayuscula, cuando las variables son constantes"
```

Formas para no crear una variable:

```python
2variable = "Numero al inicio"
mi-variable = "Con guion medio"
mi variable = "Con espacio"
```
